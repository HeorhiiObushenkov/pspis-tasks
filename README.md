# Семестровий Дашборд (ПЗПІз-25-1)

Сучасний та швидкий дашборд для організації навчального процесу студентів. Дозволяє зручно переглядати матеріали дисциплін, корисні посилання та відстежувати виконання завдань.

## ✨ Особливості
- 📊 **Зручний інтерфейс**: Побудований на базі Tailwind CSS та DaisyUI.
- 🌓 **Темна та світла теми**: Можливість персоналізації вигляду.
- ✅ **Трекер завдань**: Збереження прогресу виконання робіт.
- 📱 **Адаптивність**: Повна підтримка мобільних пристроїв.
- ⚡ **Швидкість**: Побудовано на чистому JavaScript (Vanilla JS), миттєве завантаження даних.

## 🚀 Швидкий старт
Проєкт є статичним веб-сайтом. Для коректної роботи Fetch API потрібен локальний сервер:

1. Відкрийте папку з проєктом.
2. Запустіть сервер:
   ```bash
   # Якщо у вас встановлений Python
   python -m http.server 8000
   
   # Або за допомогою Node.js
   npx serve .
   ```
3. Відкрийте `http://localhost:8000` у браузері.

## 📂 Структура проєкту
- `index.html` — основна сторінка.
- `js/` — логіка додатку та рендеринг.
- `data/` — JSON файли з контентом дисциплін.
- `style.css` — кастомні стилі.
